# MTI Event Pipeline Demo

This project is a Java/Spring Boot demo designed to simulate a production-style event processing pipeline.

The system models a simplified version of a webinar data flow:

External Webinar System → Java API → Database → CRM Sync Simulation

## Goals

- Demonstrate Java and Spring Boot fundamentals
- Show troubleshooting and debugging practices
- Document architecture and operational decisions
- Simulate legacy-style issues and progressive refactoring
- Provide a portfolio-ready project for technical interviews

## Tech Stack

- Java 17
- Spring Boot
- Maven
- PostgreSQL
- REST APIs
- Docker Compose
- Actuator
- Structured logs
